2. My Projects

Google Dino-Clone Game
A clone of the infamous Google Dino-Game. This game has functions such as jump and run in which the player will need to use in order to jump over the cactus objects. If they fail to do so, a defeated screen will appear in which the player will have the option to play the game again with any button.

Tech:HTML,CSS,JavaScript
Todo List Extensions
A todo list google extensions in which allows the user to create a todo list for the day. This todo list extension includes a cross-out feature, edit button, delete button, new day button, and a progress bar in which will be updated everytime the user completes their todo list task. Utilizing the local storage, the extension is able to keep the memory of all the task that the user has input in evem after refreshing/closing the extension.

Tech:HTML,CSS,JavaScript
Gitlet
A simple version control system that mimics the important features of Github. Some examples of the important commands that was implemented was checkout, commit, add, init, log, and etc. Also to keep and organize the correct versions of files within gitlet, I was able to serialize each file and store them in data structures such as linked lists, trees, and hash maps to log back to the versions that the users wanted.

Tech:Java, Python
